================================================================================
                            API INTEGRATION
================================================================================

ENDPOINT: POST http://localhost:5000/analyze

REQUEST BODY:
{
  "text": "User's personality text here..."
}

RESPONSE:
{
  "archetypes": ["Strategist", "Creator"],
  "traits": {
    "white": 0.7,
    "blue": 0.8,
    "black": 0.5,
    "red": 0.6,
    "green": 0.9
  },
  "entropy": 1.5
}

ERROR RESPONSE:
{
  "error": "Error message describing what went wrong"
}
